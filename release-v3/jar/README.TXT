Design Patterns — How to Run

What you need
- Java 21 installed
- Keep the entire jar/ folder

Run
Option 1 (default): Double-click the launcher
- Windows: double-click runWindows.bat
- macOS / Linux: double-click runMacLinux.sh (or right-click → Run / Open)

Option 2 (If you modify any of the files): Run from terminal / command prompt
1) Open a terminal / command prompt inside the dist/ folder
2) Run:
java -jar "Design_Patterns.jar"

Notes
- If java is not found, install Java 21 and make sure this works:
java -version
